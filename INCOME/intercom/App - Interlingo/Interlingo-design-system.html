Spacing & Layout Principles

8px Grid System

All spacing follows an 8px grid for consistency and visual rhythm:

  • 4px (space-1): Tight spacing between related elements
  • 8px (space-2): Small gaps, icon spacing
  • 16px (space-4): Standard component padding
  • 24px (space-6): Section spacing
  • 32px (space-8): Large section breaks
  • 48px (space-12): Major page sections
